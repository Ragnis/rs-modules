email.message

Represents a received or sent message.

Inherits from email.draft.

Requires the following properties to be set

  • email.draft.to,
  • email.draft.subject,
  • email.draft.body and
  • email.draft.date

Summary
Properties

Properties

name

The account owner’s name.  This name is used as the sender name for outgoing messages.

address

The address associated with this account.  Will be used as the sender address for outgoing messages.
